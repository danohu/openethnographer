# Open Ethnographer

Documentation is at https://edgeryders.eu/en/open-ethnographer/open-ethnographer-manual

Open Ethnographer is a software to add ethnographic coding right into live online content. It is an open source plug-in for Drupal, and we use it to add semantic markup to edgeryders.eu content, empowering both our ethnographic research, website visitor's navigation and discovery, and semantic search engines.
